dss_toolkit.spark.upload module

dss_toolkit.spark.upload.file_to_vault(local_file, remote_path, dv_namespace, dv_tblname, yearmo=None, file_format='csv', overwrite=False, spark_app_name='')

  1. Moves the csv/parquet file from (local) to HDFS

  2. Reads the CSV/parquet file from HDFS as spark dataframe

  3. Writes spark dataframe into the database

  4. Removes the CSV file from HDFS (undo the work done in step 1)

Parameters

  • local_file (str) – path name of csv/parquet file to upload

  • remote_path (str) – pathname in hdfs where to upload the file

  • dv_namespace (str) – datavault schema. Will be uploaded as dv_namespace.`dv_tblname`

  • dv_tblname (str) – datavault table name. Will be uploaded as dv_namespace.`dv_tblname`

  • yearmo (str) – yearmo to use as partition. If None, no partitioning will be done

  • overwrite (boolean) – whether to overwrite or append existing table

  • spark_app_name (str (default: "")) – spark session name to be used
